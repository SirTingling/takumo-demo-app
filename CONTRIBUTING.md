# Contributing

This is a reference repository demonstrating security patterns for [Takumo](https://takumo.io).

We are not accepting feature pull requests at this time. Bug reports are welcome via [Issues](https://github.com/SirTingling/takumo-demo-app/issues).

## Reporting Bugs

If you find a bug, please open an issue with:

1. Steps to reproduce
2. Expected behavior
3. Actual behavior
4. Environment details (Node.js version, OS)

## Security Issues

For security vulnerabilities, please see [SECURITY.md](SECURITY.md).
